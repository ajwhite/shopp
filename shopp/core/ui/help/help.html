Help

  • Official Documentation

    The wiki is the best place to get started learning about the Shopp e-commerce platform. As Shopp is in beta, the documentation is sketchy at the moment, but it will become more complete as it nears release.

  • Support Forums

    You can also get help from fellow Shopp owners and developers by posting your questions and problems on the public support forums. To ensure you get quick responses, be sure to completely explain your problem by providing as much information about the problem you can (which may include quoting the code you are using or the exact text of error messages you are receiving).

  • Premium Support

    Should you need in-depth assistance setting up or customizing your Shopp install, you may want to purchase premium support credit that will allow you to chat live with the Shopp developer for personal support.
